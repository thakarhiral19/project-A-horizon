# Titanic Dataset - Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

The goal is to analyze passenger data and discover patterns related to survival.

---

# Objectives

- Understand the dataset structure
- Handle missing values
- Perform statistical analysis
- Visualize distributions and relationships
- Identify important features affecting survival

---

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code
